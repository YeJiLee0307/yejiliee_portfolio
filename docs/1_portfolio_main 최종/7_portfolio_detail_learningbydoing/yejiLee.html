Making my own portfolio_SEQUENCE

I became motivated to learn HTML, CSS, and JavaScript through
platforms like YouTube and Inflearn to create a portfolio.
Through this practice, I was able to gain knowledge
about front-end development and its basic principles.

Previously, I vaguely thought that the fields of design
and development were distinct from each other and
had only an interest in design-related areas.
However, through this project,
I realized that a design-oriented mindset is essential
in front-end development.
I found it fascinating to create code
that allows me to produce my own designs,
and this experience has made me appreciate
the fusion of art and technology.

Coding class
By learning Python, which can be considered
the most basic programming language, I was able to grasp
the fundamental principles of computer languages.
I was fascinated to learn that applications and websites
like Instagram and Pinterest were built using Python code,
and I realized that with this foundational knowledge,
I could create similar applications and websites.
Through the process of solving code and debugging errors,
I became familiar with the syntax of Python.
I also learned how to visualize numerical data
as graphs and charts, which helped me understand
the principles of various diagrams used in different fields.
This experience made me aware of how crucial visual imagery is
in the process of conveying information.
 Video artwork This is the first video I created as part
of the Creativity & Visual expression class project team assignment.

of tools like Premiere Pro or After Effects,
we learned how to use these tools to produce a creative outcome.
Initially, it was challenging to convey our intentions solely
through visual imagery and effects without subtitles or audio.

explanatory and lost its appeal.
Conversely, if we presented it too abstractly, the intended message
did not come across clearly to the viewers.
